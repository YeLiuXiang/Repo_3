## 格式说明

- `[P]`：可并行（不同文件，无依赖关系时标注）
- `[US1]`/`[US2]`：关联的用户故事编号
- 无标注：依赖前序任务，必须串行执行

**DoD（完成定义）标准**：代码已提 PR + 单元测试覆盖率 ≥ 80% + PR Review 通过 + 无阻断 Bug
